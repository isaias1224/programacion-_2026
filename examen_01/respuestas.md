	EXAMEN_01.
1- Define clase.
Son los datos que opera el usuario dando una agrupación, donde incluye los datos y atributos.

2- Define encapsulamiento.
Son los datos ocultos internos de una clase que muestra o no, ciertos datos lo que integra son el privado y público.

3- Define constructor.
  Son datos que lee y se relaciona con la clase, y que ejecuta automáticamente cuando lo recrea el objetivo. lo cual el nombre de los datos no cambia.
